## 第02讲    为什么要学习积极心理学

### 一、积极心理学研究什么

```java
✔"适合"的重要性
《快乐之道》中提出"适合"的概念，不是所有的方法和实验都适合所有人，只能是多数人
判断是否适合自己是，先接触，然后尝试，最后进行判断，如果适合自己，"我要把它融入生活中";如果不适合自己，"对我没有意义"
```

```java
✔心理学消极研究：积极研究=21:1
1.消极研究：研究不起作用的东西，展示人类的短处、缺点、仅限于黑暗、恶劣、偏见、压抑、焦虑、错误、过失
2.积极研究：研究爱、目标、休息、锻炼、快乐
研究表明：近年来，抑郁症、自杀的人越来越多，我们需要克服抑郁，焦虑
47%的哈佛学生在过去一年经历过无法正常生活的抑郁期，94%的大学生感到压抑
所以，如何解决这种问题，是增大比率，研究更多的消极，还是减少比率研究更多的积极？
```

### 二、为什么是积极心理学

```java
✔我们为什么要研究积极的东西，为什么是积极心理学
reason:
	1.集中研究有用的东西至关重要，因为有用的或集中研究的会称为现实
	2.快乐并非否定痛苦，否定焦虑
	3.预防十分重要，预防焦虑、抑郁要通过专注培养积极的心态
```

```java
✔积极心理学的诞生：研究有用的东西，研究我们提出的问题，"问题缔造现实"
消极问题："为什么有这么多孩子退学？"，"为什么有这么多人失败？"
积极问题："是什么让某些人成功了？即使面对不理想的环境？"，this makes all the difference,这才是最重要的问题
当研究的问题是消极问题时，研究取得的成果微乎其微，but当研究积极问题时，研究转向成功的学生
```

```java
✔心理弹性resilience
研究结果：成功的学生拥有较好的"心理弹性"
心理弹性：积极适应，即使是面对不利或危险的环境，成功的孩子具有心理弹性，更具有适应力
```

```java
✔成功的孩子更聪明，更有天分吗？
no
成功的孩子他们并非是有天分，聪明的孩子，他们只是普通的孩子，原因是"乐观"，并非"盲目乐观"，而是对失败有积极的解读，有意义感
他们的特征：心理弹性=信心+意义感+利社会行为+帮助他人+集中自己的长处+设立目标+具有榜样+不单干（寻求支持与帮助）+承认弱点
```

### 三、问题创造现实

```java
✔提出的问题十分重要
就是因为研究提出来一个积极的问题，从而找到的正确的方向
研究疾病：致病模型
研究健康：健康模型
如果刚度完蜜月的夫妇，几个月后问"我们的关系出现了什么问题"，就很可能看不到关系中的温暖与浪漫，无法更好地发展爱情
```

```java

实验问题："屏幕上有几个几何图形"
另一个突如其来的问题："看到巴士车上的小孩了吗？"
当学生专注于第一个问题的时候，即使巴士车就在面前，也没有第二个问题的答案
所以："集中专注在一个问题容易让我们护士其他的问题，问什么就看到什么"
专注缺点就看不到优点，专注于抑郁、焦虑就看不到积极的东西，而积极的东西真是问题的答案
```

## 第03讲 幸福是一种随机现象吗

### 一、疾病模型和健康模型

```java
✔从-到+
疾病模型：-到0
健康模型：-到+
疾病模型："We are ill because we are sick",如若我们治好病，就会健康，病了就去致病
健康模型："You are ill because you don't have enough health in your life",病了要注重健康，加强健康
健康模型也就是积极心理学的思想，注重"培养自身成长的积极性"，培养良好的人际关系，培养爱情
"预防"更好的方法：疾病模型不能更好地预防焦虑和抑郁，实际上，注重培养能力，而非改变错误
```

```java
✔培养能力
培养能力：建立更强大的心理免疫系统，就算生病了也更快恢复，培养乐观、积极、追求、意义、规划等
"引擎"：我们的心理免疫系统也好比引擎，如果引擎小，要把车开上山坡容易引起损毁和爆炸，如果引擎够大，就很容易
```

```java
✔心理学研究内容
心理学（包括其他心理学）如今不仅仅对疾病、弱点和伤害进行研究，也研究优点和美德，治疗不仅仅是修复错误，同时也要塑造正确的东西，心理学关注的不仅仅是疾病和健康状况，还包括教育、洞察力、爱、成长和娱乐
```

## 第04讲 积极地环境能改变人

### 一、什么带来了改变

```java
✔改变以指数函数型发生
人若想在世间有所作为，真正实现改变，面对的最显著的障碍之一，是他们低估自己实现改变的能力
人类学家Margarent Mead:"永远不要怀疑一小群，有理想，坚定的市民可以改变世界，实际上，正是这群人改变着世界"
所有的改变从一个人或一小众人的思想开始，然后开始扩张
扩正以"指数级"发生，就像"蝴蝶效应"，粒子的相互碰撞带来无法想象的威力

社会网络的本质也是指数的，Exponetial nature of social networks
不要低估指数函数中"1"的力量，一张纸折41次可以够得到月亮
所以我们低估影响改变的能力，因为我们低估了指数函数的增长
```

### 二、外部环境的影响

```java
3.外部环境影响微乎其微
一项幸福水平的研究，审核终身职位的教授，不论是是否审核成功，半年后，他们的幸福水平，都恢复到以前，而在审核前他们会以为自已会因此快乐一辈子，或悲伤一辈子
这项研究的结论是："极端情况对我们的幸福影响很少"
在基本生存需求满足后，收入对幸福的影响微乎其微，我们比祖父辈富裕，但是我们并没有比他们更幸福，如果基本生存没有得到满足，例如一个天天被人欺负的乞丐，一天他富裕了，幸福水平就会有很大的提升
幸福水平几乎是静止的
就像被被哈佛录取后喜出望外的学生，一段时间后，他们的幸福水平会渐渐恢复平均

```
