## 第02讲    为什么要学习积极心理学

### 一、积极心理学研究什么-为什么学习积极心理学

```java
✔"适合"的重要性
《快乐之道》中提出"适合"的概念，不是所有的方法和实验都适合所有人，只能是多数人
判断是否适合自己是，先接触，然后尝试，最后进行判断，如果适合自己，"我要把它融入生活中";如果不适合自己，"对我没有意义"
```

```java
✔心理学消极研究：积极研究=21:1
1.消极研究：研究不起作用的东西，展示人类的短处、缺点、仅限于黑暗、恶劣、偏见、压抑、焦虑、错误、过失
2.积极研究：研究爱、目标、休息、锻炼、快乐
研究表明：近年来，抑郁症、自杀的人越来越多，我们需要克服抑郁，焦虑
47%的哈佛学生在过去一年经历过无法正常生活的抑郁期，94%的大学生感到压抑
所以，如何解决这种问题，是增大比率，研究更多的消极，还是减少比率研究更多的积极？
```

### 二、为什么是积极心理学-为什么学习积极心理学

```java
✔我们为什么要研究积极的东西，为什么是积极心理学
reason:
	1.集中研究有用的东西至关重要，因为有用的或集中研究的会称为现实
	2.快乐并非否定痛苦，否定焦虑
	3.预防十分重要，预防焦虑、抑郁要通过专注培养积极的心态
```

```java
✔积极心理学的诞生：研究有用的东西，研究我们提出的问题，"问题缔造现实"
消极问题："为什么有这么多孩子退学？"，"为什么有这么多人失败？"
积极问题："是什么让某些人成功了？即使面对不理想的环境？"，this makes all the difference,这才是最重要的问题
当研究的问题是消极问题时，研究取得的成果微乎其微，but当研究积极问题时，研究转向成功的学生
```

```java
✔心理弹性resilience
研究结果：成功的学生拥有较好的"心理弹性"
心理弹性：积极适应，即使是面对不利或危险的环境，成功的孩子具有心理弹性，更具有适应力
```

```java
✔成功的孩子更聪明，更有天分吗？
no
成功的孩子他们并非是有天分，聪明的孩子，他们只是普通的孩子，原因是"乐观"，并非"盲目乐观"，而是对失败有积极的解读，有意义感
他们的特征：心理弹性=信心+意义感+利社会行为+帮助他人+集中自己的长处+设立目标+具有榜样+不单干（寻求支持与帮助）+承认弱点
```

## 第03讲 幸福是一种随机现象吗-改变可能吗？

### 一、疾病模型和健康模型-幸福是随机的吗

```java
✔从-到+
疾病模型：-到0
健康模型：-到+
疾病模型："We are ill because we are sick",如若我们治好病，就会健康，病了就去致病
健康模型："You are ill because you don't have enough health in your life",病了要注重健康，加强健康
健康模型也就是积极心理学的思想，注重"培养自身成长的积极性"，培养良好的人际关系，培养爱情
"预防"更好的方法：疾病模型不能更好地预防焦虑和抑郁，实际上，注重培养能力，而非改变错误
```

```java
✔培养能力
培养能力：建立更强大的心理免疫系统，就算生病了也更快恢复，培养乐观、积极、追求、意义、规划等
"引擎"：我们的心理免疫系统也好比引擎，如果引擎小，要把车开上山坡容易引起损毁和爆炸，如果引擎够大，就很容易
```

```java
✔心理学研究内容
心理学（包括其他心理学）如今不仅仅对疾病、弱点和伤害进行研究，也研究优点和美德，治疗不仅仅是修复错误，同时也要塑造正确的东西，心理学关注的不仅仅是疾病和健康状况，还包括教育、洞察力、爱、成长和娱乐
```

## 第20讲 幸福与幽默-幽默

```java
✔幽默对心理的益处
1.幽默是一种心理释放
弗洛伊德写过一本书《幽默及其潜意识的关系》，他在书中描述，意识在我们的内心有一个本我，本我包含了大家都有的性欲、生命力和冲动；本我之外是自我，这个自我会把本我视为本我冲动和社会之间一道可渗透的屏障；最外面的是超我，超我是加在我们身上的道德约束和社会约束
如果我的本我表示"我很生气，我想揍你"，超我的道德约束就会阻止我打你。我们都会受挫，因为我们都有本我冲动，但我们没有办法把这些冲动释放出来，所以我们就会被压抑，最终就会爆发。
弗洛伊德认为幽默是一种社会可接受的，用来疏导这些欲望的信封，是一种把本我冲动释放出来的方法
```

```java
根据自我知觉理论，如果你想在某个生活领域得到改善，最好的方法是让自己的行为变得像改善了一样，如果我想要高度自尊，我的行为就要像一个高度自尊的人，因为自尊是一种态度，是我对自己的评价，要提高这种态度，最有效的方法就是表现出一个有高度自尊人的行为
1.更平静
行为平静的人他们的独立型自尊会随着时间增加
练习正念冥想，我们的独立型自尊能够加强，冥想的重点是带来平静，关注自己，活出自己
运动能带来平静，它在体内释放某种化学物质，能够直接增强我们的独立性自尊
2.找到自己的激情
想象在一个世界中，没有人知道你做什么，除了你自己，没有人知道你多好，没有人评价你。你真正想做的是什么？你所热爱的是什么？你想要怎样生活？
3.发现心流
有高度独立型自尊的人会追求他们的兴趣，独立型自尊也会增加。"你什么时候忘我过?什么时候体验过心流?读书的时候吗？是写作室吗？"
4.勇于应对
有高度独立型自尊的人勇于付诸行动，不断加强自己，总是学习、成长、进步，触及更多的人
5.谦虚
"那些追逐荣誉的人，荣誉会躲避他们，那些躲避荣誉的人，荣誉会追逐他们。"荣誉的本质是尊敬自己，相信自己，如果我们不断追求下一个荣誉，追逐下一个赞扬，下一个认同，荣誉就会躲避我
相反，如果我躲避荣誉言行谦逊，随着时间过去，真正的荣誉就会追逐我
6.慢慢来
为什么在美国这样一个国家里，一个追逐个人主义，自力更生的国家，有那么多关于个人主义和自我的书，为什么很多人还是那么随波逐流？
我想这可能是因为我们没有花足够的时间来反省，反省我真正想过的人生是怎样的。我们退后一步，慢慢思考才能变成自我，才能找到适合自己的道路
研究发现，当人们的认知忙乱时，他们的行为会不同于当他们慢慢花时间专心思考时所做的事
关于时间有三个阶段：
第1个阶段是我们没有时间的阶段，很赶时间，匆匆忙忙，反应这时的反应是无意识的，我们会随波逐流，遵从群体，这是依赖的自我
第2阶段，如果我们慢慢来，这时我们就有意识地去思考吧，思考与行为结合起来，会想到自己想做的事情，We take our time，这个阶段的自我是自主的，独立型的自尊就出现了，这使我们的随波逐流的可能性降低
本课程最重要的部分是课堂及课后作业，因为在课后我们有时间思考、讨论，这时我们真正的自我就浮现了
7.正直
正直是指言行一致，如果言行不一致，那我们本质是在对自己说:"我说得话无关紧要不重要",反之如果我保持高度正直，遵守诺言，根据自我知觉理论，通过行动我会对自己说:"我说的话很重要，我很重要"，所以要正直
```

```java
这是Tal对本课程最重要的11点的总结：

允许为人，世界上有两种人不会经历情感创伤，精神病患者和死人
当情感当经历感情创伤时，你会对着他说，我只是普通人，我很难过，但我接受它，就像接受重力定律一样，因为重力定律是一种物理本质，就像感情创伤是一种人性本质。允许为人

写日记表达情绪，当我们分享开放内心时，就会建立一种关联性
6.Cultivate the benefit finder (express gratitude)
感激能够得到更高层次的快乐，感激生活中好的方面，感激人际关系中好的方面，感激我们的伴侣，朋友，同学家人，感激我们的国家
感激它们，这些好处就会增值，当我们不以感激时，好的事物也会贬值
7.Simplify
说起来容易做起来难，时间充裕能够带来幸福，果断坚决，在适当的时候学会说"不",看清楚你真正想做的事情，然后去做
8.Cultivate relationships
最能预示幸福的是人际关系,亲密关系比世界任何事情都重要,甚至比问问题更重要，比考试更重要
成功使幸福感突升，然后又很快降到基础水平，但亲密关系却能维持幸福，经营一段感情意味着为之投入，投入时间，彼此分享和交流
9.Remember the mind-body connection
精神与肉体相互联系，基因决定基准幸福水平，当我们不锻炼时，提升我们的幸福水平是很难的
去做做瑜伽或冥想、每天深呼吸几次、每天尽可能8小时、睡眠以及触摸、拥抱
10.Strive to be known rather than validated
被了解而不是被认可，无论是和我们的伴侣、家人、朋友、同事、同学，这都是在长期人际关系中亲密和激情增加的基础
11.Introduce behavioral change NOW
转变，马上行动，上完幸福课之后能否提高幸福感，取决于你是否发生了行为转变，"不要告诉我这课有多棒，告诉我你周一做了什么改变"，从现在开始转变，这就是改变的核心
```
